

# إضافة فلاتر إضافية لقائمة الزوار

## الخطة

إضافة شريط فلاتر ذكي أسفل تبويبات الحالة (الكل/متصل/غير متصل) يتضمن فلاتر إضافية:

### الفلاتر المقترحة
1. **الدولة** - قائمة منسدلة بالدول المتاحة من بيانات الزوار
2. **الجهاز** - موبايل / ديسكتوب
3. **المتصفح** - Chrome / Safari / Firefox / Edge
4. **لديه طلبات معلقة** - زر تبديل لإظهار الزوار الذين ينتظرون إجراء فقط

### التصميم
- شريط أفقي قابل للتمرير بأزرار chips صغيرة ومدمجة
- كل فلتر نشط يظهر بلون مميز مع إمكانية إزالته بنقرة
- زر "مسح الكل" لإعادة تعيين جميع الفلاتر

### التعديلات التقنية
- **ملف واحد**: `src/pages/admin/AdminVisitors.tsx`
  - إضافة state جديد `filterBrowser` و `filterHasPending`
  - المتغيرات `filterCountry` و `filterDevice` موجودة فعلاً في الكود
  - إضافة واجهة الفلاتر بعد تبويبات الحالة مباشرة
  - تحديث منطق `filtered` لتشمل الفلاتر الجديدة
  - تصميم chips مدمج يتناسب مع التصميم الحالي (rounded-xl، ألوان متدرجة)
